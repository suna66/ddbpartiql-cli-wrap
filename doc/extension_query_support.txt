# Describe table query

## query format

```
desc "{table name}";
```

# Create table query

## query format

```
create table "{table name}" (
    {fieldName} {N|S|B} [HASH/RANG],
    {fieldName} {N|S|B} [HASH/RANG]
     :
    [index {local|global} {index name} (
        {fieldName} {N|S|B} [HASH/RANG],
        {fieldName} {N|S|B} [HASH/RANG]
    )]
);
```

# Delete table query

## query format

```
drop table [if exists] "{table name}";
```
